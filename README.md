# TE Communications — Daily Media Intelligence Agent

Tägliches Finanzmarkt-Morning-Briefing für TE Communications GmbH.

## Scope

Breite, quellenbasierte Medienbeobachtung und analytische Einordnung über **12 Themenfelder**:

| # | Themenfeld |
|---|---|
| 1 | Makroökonomie & Konjunktur |
| 2 | Geopolitik & Sicherheitspolitik |
| 3 | Energie & Rohstoffe |
| 4 | Zentralbanken & Geldpolitik |
| 5 | Aktienmärkte |
| 6 | Anleihemärkte & Fixed Income |
| 7 | FX / Devisen |
| 8 | Private Credit / Private Markets |
| 9 | Emerging Markets |
| 10 | Krypto / Digital Assets / Tokenisierung |
| 11 | Immobilienmärkte (gewerblich & Wohnimmobilien) |
| 12 | ESG / Sustainable Finance / Regulierung |
| + | M&A / Deals / IPOs im Asset Management |

## Einordnungslinse

Die Berichterstattung wird thematisch breit recherchiert. Am Ende jeder Tagesauswertung erfolgt eine gezielte Einordnung für:

- **PGIM** (Prudential Financial, $1,5 Bio. AuM)
- **T. Rowe Price**
- **MK Global Kapital** (Impact-Fonds ALTERNATIVE, Luxemburg)
- **Franklin Templeton** ($1,74 Bio. AuM)
- **PIMCO**
- **Eurizon** (Intesa Sanpaolo)

## Quellenspektrum

Reuters, Handelsblatt, FAZ, FT, Bloomberg, ZDF, Börsen-Zeitung, SZ, WiWo, Spiegel, Manager Magazin, Fonds Professionell, Citywire, DAS INVESTMENT, finanzen.net, IPE, IEA, Fortune, Axios, NPR, CNBC, CoinDesk, CBRE, Cushman & Wakefield, Deloitte, JLL, Raiffeisen Research, dpa-AFX u.v.m.

## Struktur der Tagesauswertung

```
1–N.  Nummerierte analytische Themenblöcke (nach Relevanz)
N+1.  Konkrete Einordnung pro Haus
N+2.  Was die Berichterstattung NICHT dominiert
N+3.  Finanz- & Kapitalmarkttermine (7-Tage-Kalender)
      Verdichtetes Fazit
```

## Output

Interaktives React-Dashboard (`.jsx`) mit:
- KPI-Leiste (Brent, Gold, DAX, Nikkei, Rupie, Fed Rate, Bitcoin etc.)
- Akkordeon-Blöcke pro Thema
- Alert-Box bei Eskalationen
- Quellennachweis

## Zeitplan

- **Täglich 07:00 Uhr CET** — Morning Briefing
- Recherche erfasst die bis dahin indexierte Berichterstattung

## Nutzung

Den Agent als React-Artifact in Claude.ai öffnen. Für die tägliche Aktualisierung den Morning-Briefing-Prompt mit aktuellem Datum absenden.
